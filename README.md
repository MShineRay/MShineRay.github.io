## env
- 配置hosts
  ```
  host文件位置：
  mac：sudo vim /etc/hosts
  
  windows：c:\windows\system32\drivers\etc
  
  
      #FE localtion  environment
      127.0.0.1 mock.local.fe.cn
      127.0.0.1 dev.local.fe.cn 
      127.0.0.1 test.local.fe.cn
      127.0.0.1 sit.local.fe.cn

- Project setup
  ```
  npm install
